# Condicionais e Laços

## 1. Condicionais

A sintaxe se por:

`if condicao then valor_se_verdadeiro else valor_se falso`

Onde:

- `condicao`é um bool.
- valores tem o mesmo tipo.
- `else`é obrigatório.

**Exemplo:**

```bash
ghci> x = 6
ghci> resultado = if x >= 5 then "Aprovado" else "Reprovado
"
ghci> resultado
"Aprovado"
```
