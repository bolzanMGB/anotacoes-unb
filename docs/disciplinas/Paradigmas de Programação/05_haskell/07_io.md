## 2. Manipulação de Arquivos

**1. Abrir arquivo**

`variavel <- openFile "nomeArquivo.txt" modo`

- Ele entrega o arquivo com um cursor invisível para primeira linha
- Os modos podem ser: ReadMode, WriteMode, AppendMode e ReadWriteMode.


**2. Fechar Arquivo**

`hClose variavel`

**3. Handle Is End Of File**

`variavelBoolean <- hIsEOF variavelArquivo`

Retorna se um boolean chegou ao fim do arquivo ou não.

**4. Handle Get line**

`linha <- hGetLine variavelArquivo`

- Ele lê todo o texto da linha onde o cursos está parado e joga na variável.
- Ele arrasta o cursos automaticamente para o início da próxima linha.

**5. Handle Get Contents**

`variavel <- hGetContents variavelArquivo`

- Pega todo o contedúo do arquivo a partir de onde está o cursor de leitura.


**6. Handle Put String Line**

`hPutStrLn variavelArquivo line`

- Coloca a linha em outro arquivo.


**Exemplo**

Ele vai pegar o input.txt, formatá-lo (retirar os espaços desnecessários) e colocar dentro do output.txt.


```haskell
import System.IO

main = do
    inh <- openFile "input.txt" ReadMode
    outh <- openFile "output.txt" WriteMode
    mainLoop inh outh
    hClose inh
    hClose outh

mainLoop inh outh = do
    ineof <- hIsEOF inh
    if ineof then return ()
        else do
            line <- hGetLine inh
            hPutStrLn outh $ stripped line
            mainLoop inh outh

 stripped = unwords . words
```


**3. Controlar Cursor de Leitura**

**1. hTell**

`size <- hTell variavelArquivo`

- Retorna a posição atual do cursor do arquivo, contada em número de bytes, em uma variável.

**2. hSeek**

`hSeek variavelArquivo Modo Numero Posicao`

- Atualiza a posição do cursor.
- Pode ser AbsoluteSeek, RelativeSeek, SeekFromEnd

**Exemplo**

Computa o número de bytes contidos no arquivo indicado

```haskell
import System.IO

main = do
    putStrLn "Insira o nome do arquivo:"
    path <- getLine
    h <- openFile path ReadMode
    size <- fileSize h
    print size
    hClose h

fileSize :: Handle -> IO Integer
fileSize h = do
    hSeek h SeekFromEnd 0
    size <- hTell h
    return size
```

## 3. Remover e Renomear Arquivos

**1. removeFile**

`removeFile "caminho/do/arquivo/nome.formato"`

**2. renameFile**

`renameFile "caminho/do/arquivo/nome.formato" "caminho/do/arquivo/novoNome.formato"

**Exemplo**

```bash
import System.Directory (removeFile, renameFile)

main :: IO ()
main = do
    putStrLn "A limpar o sistema... A apagar o rascunho."
    removeFile "rascunho.txt"
    putStrLn "O ficheiro 'rascunho.txt' foi removido com sucesso!"

    putStrLn "A finalizar o processo... A renomear o resultado."
    renameFile "resultado_provisorio.txt" "relatorio_final.txt"
    putStrLn "O ficheiro foi renomeado para 'relatorio_final.txt'!"
```

## 4. Lazy I/O

**1. readFile**

`variavel <- readFile "nomeArquivo.formato"`

- Você passa o nome do arquivo e ele te devolve o conteúdo inteiro dentro de uma String.

**2. writeFile**

`writeFile "nomeArquivo.formato" variavelQueVaiColocarDentro`

- Você passa o nome do arquivo e ele escreve dentro dele.

**3.interact**

`interact funcao1 . funcao2`

- Faz todo o processo da saída e entrada padrão (via terminal) automaticamente
