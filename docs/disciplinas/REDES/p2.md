1. Servidor DHCP
- Dynamic Host Configuration Protocol
- Protocolo de rede para atribuir endereços IP e outras configurações aos dispositivos conectados a uma rede

2. Servidor DNS
- Domain Name System
- Serve para traduzir nomes de domínios em endereços IP

2. VLAN
- Virtual Local Area Network
- Serve para dividir logicamente uma rede física em múltiplas redes menores (subredes).
- Separa grupos de dispositivos conenctados fisicamente no mesmo switch
